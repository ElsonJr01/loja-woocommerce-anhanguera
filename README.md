# Desenvolvimento de Loja Virtual com WooCommerce

Projeto acadêmico desenvolvido para a disciplina **Desenvolvimento de E-commerce com CMS**, da Faculdade Anhanguera.

## 1. Identificação
- **Instituição:** Faculdade Anhanguera  
- **Disciplina:** Desenvolvimento de E-commerce com CMS  
- **Unidade:** U3 – E-commerce com WooCommerce  

## 2. Objetivo do Projeto
Desenvolver uma loja virtual funcional utilizando um Sistema de Gerenciamento de Conteúdo (CMS), por meio da plataforma WordPress integrada ao plugin WooCommerce, possibilitando o cadastro de produtos, configuração de métodos de pagamento e entrega, além da personalização da página inicial da loja.

## 3. Tecnologias Utilizadas
- WordPress  
- WooCommerce  
- PHP  
- MySQL  
- XAMPP  
- Git e GitHub  

## 4. Metodologia
O desenvolvimento do projeto foi realizado em ambiente local, utilizando o pacote XAMPP para simulação de um servidor web. Inicialmente, foi criado um banco de dados para armazenamento das informações da loja. Em seguida, o WordPress foi instalado e configurado, juntamente com o plugin WooCommerce, responsável pela implementação das funcionalidades de comércio eletrônico.

Posteriormente, foram realizados o cadastro de produtos e categorias, a configuração de métodos de pagamento e entrega, além da personalização visual da loja. Por fim, o projeto foi versionado utilizando a plataforma GitHub, seguindo boas práticas de segurança da informação.

## 5. Versionamento e Segurança
Este repositório contém apenas a estrutura do projeto WordPress. Arquivos sensíveis, como configurações de banco de dados (wp-config.php), banco de dados e arquivos de upload, não foram versionados, visando a segurança das informações.

## 6. Estrutura do Projeto
- `wp-admin/` – Painel administrativo do WordPress  
- `wp-content/` – Temas, plugins e arquivos do projeto  
- `wp-includes/` – Arquivos principais do WordPress  

## 7. Observações Importantes
Este projeto tem finalidade exclusivamente acadêmica. Para execução completa da aplicação, é necessário configurar um ambiente local ou servidor web com suporte a PHP e MySQL, além da criação de um banco de dados próprio.
